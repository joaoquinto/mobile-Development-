# Tipografia 

## Tipos de Fontes

### Sem Serifas 

- São neutras, funcionam bem em corpo de texto, usadas frequentemente na web para compor o texto (Helvetica, Arial, Avenir).

### Serifadas

- São clássicas, funcionam bem em corpo de texto, usadas em livros para compor o texto (Garamont, Georgia, Baskerville).

### Serifas grossas

- Encorpadas, funcionam bem em titúlos (Rockwell, Bitter e Kreon).

### Script

- Suaves, parecem ter sido feitas a mão e com um pincel. Funcionam bem em titúlos (Salamander, Lucida Calligraphy e Brush Script).

### Góticas/ Blackletter

- Estilizadas, utilizadas como referência ao período medieval. Boas para compor titúlos (New Rocker, Lucida Blackletter e Cabazon).

### Display 

- Estilizadas , não existe padrão definido, são criadas de acordo com o projeto. Cada uma possui personalidade própria. Boas para compor titúlos (Phophate, Curlz MT e Hobo).

## Variáveis Tipográficas 

- Tamanho do tipo 
    - Ex: 14px, 16px ou 18px.
- Altura da linha (Distância entre a primeira linha e a segunda em diante.)
    - Ex: 1,5 x Tamanho do Texto. 
    - Multiplicar a const 1,5 pelo tamanho da fonte(1,5 x 14px = 21px ).
- Largura do Texto
    - Máximo de 10/14 palavras por linha. 

    ### Obs: Indicações feitas para corpo de texto utilizando tipografias como helvetica e arial (Cada tipo de fonte tem a sua manipulação de variável)

## Escalas Tipográgicas (Definição prévia do tamanhos da fonte.)

- Utilize no máximo 6 tamanhos
    - Utilizar mais de 6 tamanhos diferente de texto pode criar um ruído visual.
- Escolha uma escala harmônica
    - A que eu mais utilizo é 11px, 14px, 18px, 24px, 36px e 48px
- Contraste é rei
    - Nunca utilize tamanhos próximos, como 14px e 15px. Utilizar tamanhos diferentes serve exatamente para diferenciar uma informação da outra.

## Pareando Tipos

- Contraste é rei
    -  Nunca use tipografias parecidas como helvetica e arial para criar uma composição.
- Use no máximo 2
    - Quebre a regra apenas quando você dominar ela.
- Escolha tipos complementares
    - Tipografias serifadas para o titúlo e sem serifas para o corpo de texto podem ser uma boa combinação.

## Alinhamento

- À esquerda
    - Esse é o padrão web, você vai usar 80% das vezes.
- Centralizado
    - Bom para titúlos, subtítulos ou pequenos blocos de texto.
    - O layout deve acompanhar este alinhamento.
- Justificado
    - Pouco usado na web, pois ainda não existem meios fáceis que permitam a hifenização do texto. Quase sempre gera caminhos de rato na web.

## Elementos Hierárquicos

- Negrito
    - Utilizado para destacar palavras importantes
- Itálico
    - Utilizado para estilizar citações, títulos de livros/filmes e outros elementos.
- Sublinhado
    - Geralmente utilizado para destacar palavras que possuem link para outra página.

## Elementos

- Caixa Alta
    - Utilizado principalmente em títulos/subtitulos.
    - Nunca utilize para corpo de texto.
    
    ### obs: Nunca utilize mais de um elemento para destacar uma palavra. Uma palavra em negrito no meio de um texto rugular, já é destacada o suficiente. Não é necessário utilizar negrito, itálico e sublinhado juntos.

## Tracking & Kerning

- Tracking é a distância entre todas as letras.
- Kerning é a distância entre duas letras.
