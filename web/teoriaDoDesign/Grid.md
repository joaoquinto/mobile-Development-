# Grid

- O grid é uma abordagem para a organização de imagens, campos de texto, títulos e dados, afim de transmitir uma informação.

## Grid na Web

- Tamanho da coluna
    - É importante existir uma lógica e ser consistente.
- Espaçamento entre colunas
    - Deve ser padronizado. Não importa se é valor fixo ou relativo.
- Encaixe os elementos na grid
    - Quebre apenas quando necessário

##  Grid Responsivo

- Tamanho da coluna adaptável
    - Pode diminuir ou aumentar de acordo com o tamanho do suporte da (tela)
- Coluna móveis
    - Colunas que ficam lado a lado em uma tela maior, podem passar a ficar uma embaixo da outra em uma tela menor.
